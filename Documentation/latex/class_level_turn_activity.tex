\doxysubsection{Detailed Description}
Level turn training activity controller. 

Manages a progressive banking exercise where users practice aircraft turns at increasing bank angles (15, 30, 45, 60) followed by a stall recovery demonstration. Implements the \doxylink{interface_i_activity_controller}{IActivity\+Controller} interface for integration with the \doxylink{class_module_activity_scheduler}{Module\+Activity\+Scheduler} system.

Training sequence\+:
\begin{DoxyEnumerate}
\item Bank -\/ Practice banking at progressive angles with VR controller input
\item Stall Setup -\/ Automatically position aircraft to 60 bank
\item Stall -\/ Practice pitch control to induce controlled stall
\item Complete -\/ Wait for activity termination
\end{DoxyEnumerate}
