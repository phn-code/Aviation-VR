\doxysubsection{Detailed Description}
Static helper class for activity management and control. 

Provides utility functions for\+:
\begin{DoxyItemize}
\item Timeline and Playable\+Director management
\item VR controller input processing and normalization
\item Aircraft rotation control (pitch and roll)
\item Debug mouse controls for testing
\item Rotation clamping and target verification
\end{DoxyItemize}
